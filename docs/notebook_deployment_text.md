The saved 2026 NBA Draft predictions are deployed as a lightweight static GitHub Pages dashboard in the repository `docs/` folder. The page reads a generated JSON file built from the existing model board and ESPN mock-draft comparison, so it can be shared without rerunning the notebook or hosting a server. After the real 2026 NBA Draft, adding `data/final/actual_draft_2026.csv` and rerunning the dashboard build script will update the page with actual evaluation metrics and visualizations without retraining the model.
